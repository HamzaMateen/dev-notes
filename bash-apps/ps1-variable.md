# PS1 Variable

## Directory

| Symbol | Meaning                       |
|--------|-------------------------------|
| `\W`   | basename of current directory |
| `\w`   | path to current directory     |

## Machine and user

| Symbol | Meaning                       |
|--------|-------------------------------|
| `\h`   | hostname                      |
| `\H`   | full qualified domain name    |
| `\u`   | username                      |
| `\$`   | `#` if root, `$` otherwise    |

## Date and time

| Symbol | Meaning                          |
|--------|----------------------------------|
| `\@`   | 12hr time (HH:MM AM|PM)          |
| `\T`   | 12hr time w/ seconds (HH:MM:SS)  |
| `\A`   | 24hr time (HH:MM)                |
| `\t`   | 24hr time w/ seconds (HH:MM:SS)  |
| `\d`   | date like _Mon May 20_           |

## Shell

| Symbol | Meaning                       |
|--------|-------------------------------|
| `\s`   | shell                         |
| `\v`   | shell version                 |
| `\V`   | shell release                 |
