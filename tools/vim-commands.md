# vim commands

| command | description |
|---------|-------------|
| :wq | write and close file |
| :q | close file |
| :q! | force closing file |
| :10 | go to 10th line |
| a | insert text before the cursor |
| i | insert text after the cursor |
| dw | delete word |
| dd | delete line |
| u | undo |
| p | paste deleted text after the cursor |
| P | paste deleted text before the cursor |
